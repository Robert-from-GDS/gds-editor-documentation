Introduction to GDS Editor

GDS Editor is a versatile web based HTML tool for creating, editing, and composing game assets. It is designed to help developers take full control of their assets with ease. The goal of GDS Editor is to bring together the most common and essential image editing needs for game development into one streamlined, easy-to-use workspace.

Input/output

Input: You can import PNG, JPG,.SVG .GDS save files, text and SVG path Data, font formats such as OTF, WOff TTF, and SCML into the editor

Output: GDS editor outputs in High resolution uncompressed PNG ready for inclusion directly into your project.

GDS Editor Core vs. GDS Editor Plus

GDS Editor Core is the base version of the application, packed with powerful features and tools available to everyone. No signup or login is required to use the core editor, making it simple and accessible right from the start.

GDS Editor Plus is the extended, supporter version of the editor. It provides additional features for users who wish to support the Game Developer Studio project. Plus unlocks unlimited access to a growing library of directly importable assets, backgrounds, textures, graphical interface modules, and other creative resources designed to accelerate your game development.
