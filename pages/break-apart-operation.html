Break Apart Operation

The Break Apart operation allows you to take any visual component and separate it into individual “pixel islands.” A pixel island is a visually isolated group of pixels surrounded by transparent space (alpha). A common example is a texture atlas, where multiple graphics are packed into a single transparent PNG image.

Accessing Break Apart

The Break Apart operation can be performed on any visual component through the right-click context menu. Select a component, then right-click on it to open the component menu. When only one component is selected, the Break Apart option will appear in the menu.

Because Break Apart is a heavy processing operation, it can only be applied to a single component at a time. If multiple components are selected, the option will not appear in the context menu.

How Break Apart Works

Break Apart analyzes the selected component and separates its pixel islands into new independent components. All resulting components are created as type "png". Only pixels that belong to each island are preserved — even if other pixels exist within the bounding box, they are clipped away.

Break Apart currently does not apply any alpha threshold. Any pixel with non-zero opacity (even 0.001) is considered part of an island.

When the operation completes, the new components are added to the scene and automatically named using the parent component's name with a child number appended.

If no pixel islands are detected, the operation will safely cancel without creating any new components.

If pixel islands appear to be connected due to semi-transparent pixels, you can use the Scalpel tool to manually remove unwanted pixels before performing Break Apart.
