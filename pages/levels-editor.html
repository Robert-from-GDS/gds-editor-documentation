Levels Editor

The Levels Editor

The Levels editor remaps pixel brightness values to change contrast, shadows, midtones and highlights. It lets you set input black/white (which clip existing values), adjust the gamma (midtones), and set output black/white (final remapped range). This is a fast and powerful way to improve image contrast, fix exposure and do basic color corrections.

Levels operates on pixel data — it edits the raster representation of an image. Using levels on vector based components will not bake in color changes to the underlying xml. Levels is available to use on vecotrs but will not be carried into the vetor data when exporting SVG data.

How to open the Levels Editor

The Levels dialog becomes available from the component inspector options panel on the right side of the workspace when a single image component is selected. component inspector #10

Once open you can start editing the highlights and shadows of your selected component via the levels dialog

Slider descriptions — what each control does

Input controls

Black InputSets the darkest input value. Pixels below this become pure black — increases shadow contrast.
White InputSets the brightest input value. Pixels above this become pure white — increases highlight contrast.
Gamma / MidtonesAdjusts midtone brightness without crushing blacks or whites. Higher gamma darkens midtones, lower brightens them.

Output controls

Black OutputLimits how black the darkest pixels can be (raise for faded look).
White OutputLimits how bright highlights can be (lower to reduce harsh whites).
All sliders update the histogram and the preview in real time. Use small steps for subtle changes and check the histogram to avoid clipping important details.

Understanding channels

The Levels dialog supports channel selection: RGB (all channels together), or individual channels R, G, B.

  • RGB: remaps brightness across all channels — overall contrast/brightness adjustment.
  • R: remaps the red channel only — useful for warming or cooling an image, or fixing red casts.
  • G: remaps green values only — useful for neutralising green tints or enhancing foliage tones.
  • B: remaps blue values only — useful to adjust coolness or remove blue casts.

Per-channel edits let you target color casts and tint shifts without affecting the overall luminance shape handled by RGB Levels.

Applying Levels to multiple components

To avoid blocking the main thread with heavy image processing, Levels only edits one component at a time in the dialog. However, you can copy and paste Levels settings to other components quickly:

  1. Adjust Levels on one image.
  2. Click the Copy style icon at the top of the Levels dialog.
  3. Select another bitmap or SVG component and press Alt + V to paste the style.

This applies the same numeric Levels settings to the target component without opening the dialog for each item—fast and convenient for batch consistency.

Understanding the histogram

The histogram shows how many pixels exist at each brightness level:

  • Left = shadows (blacks)
  • Middle = midtones (grays)
  • Right = highlights (whites)

Tall spikes show concentration at that tonal value. When you move the input black/white sliders, the histogram helps you see clipping (sudden loss of detail) and whether you are stretching or compressing tonal ranges.

A gradient strip under the histogram shows the current mapping from input → output and helps visualize how tones are remapped across the range.

Applying Levels to non-raster components

Levels requires pixel data. To apply Levels to non-raster content (paths, panels, tilesets, texture fills), convert or flatten the object to a raster image:

  1. Duplicate the component you want to preserve.
  2. With the duplicate selected, press Ctrl + F to Flatten into a raster component.
  3. Open Levels on the flattened copy and apply adjustments.

Flattening creates a pixel snapshot of the vector/frame and is reversible only if you keep the original component. It's always better to duplicate first.

Tips & best practices

  • Always keep a copy of the original (or use the component's baseCanvas) so you can revert adjustments easily.
  • Use small gamma changes and watch the histogram to avoid crushed shadows or blown highlights.
  • When working on icons or pixel-art, disable smoothing (image smoothing) to retain crisp edges. Flatten at the native size before editing.
  • Use style copy/paste (Alt + V) to quickly standardize levels across multiple assets.
