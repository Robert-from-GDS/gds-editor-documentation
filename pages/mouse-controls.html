Mouse Controls

Left Mouse Button

Mouse Action Description
left click + drag on empty scene area Pan around the scene
left click on component Select that component and unselect others
left click + shift on component Add component to selection (multi-select)
left click + shift on already selected component Remove component from selection
left click + control on empty scene area Drag out a rubber band selection rectangle
left click + drag selected components Move components around the scene
left click + drag + alt on selected components Drag along a single axis (X or Y)
left click on empty scene area Unselect all selected components

Right Mouse Button

Mouse Action Description
right click on empty scene area Open right-click context menu for scene controls
right click on single component Open right-click action menu for single component
right click on two single components Open right-click action menu for two components
right click on multiple components Open right-click action menu for multiple components

Mouse Wheel

Mouse Action Description
mouse wheel up Zoom in
mouse wheel down Zoom out
