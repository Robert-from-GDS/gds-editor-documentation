Saving a Scene

You can save a scene at any time during your workflow. Saving is simple, and there are several methods:

üíæ Saving a Scene

  • Use the hotkey Control + S to save a .gds file on your local disk. Unless you rename the save destination, the file name will match your scene name.
  • Use the quick action icon at the top of the Scene Palette to save your scene.
  • Via the right-click context menu by right-clicking on the scene area with no components selected. Mouse control instructions
  • Via the Scene Settings menu.
Currently, GDS Editor does not have an autosave feature, so be sure to save regularly.

üìÑ GDS File Format

Only GDS Editor can read and parse the .gds file format. This means you cannot drop .gds files into other editors such as Photoshop or Illustrator. If you wish to use your work in those programs, you will need to export all components as PNGs and load them as layers.

Under the hood, the .gds file is just structured JSON data, so it might be possible in the future to build a converter.

üóÇ Saving in Other File Formats

Currently, GDS files can only be saved and exported as .gds files. It is not possible to save the entire scene in other formats such as .psd, .svg, or .pdf. You can, however, export individual components in their respective formats.

In the future, I may add support for exporting whole scenes in other formats, but this is currently a low priority.
