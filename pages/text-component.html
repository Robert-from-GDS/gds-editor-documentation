Text Component User Guide

This guide explains all the properties and functions available to the user when interacting with a Text Component in the editor.

Text Input

Property Description / Action
Text Area Edit the text content of the component. Changes are applied immediately on input and logged for undo.
Font Family Select a font from the dropdown to change the text's font. Changes apply to selected components.

Text Formatting

Action Description
Bold Toggle bold on/off for the selected text.
Italic Toggle italic on/off for the selected text.
Uppercase Convert all text in the component to uppercase.
Lowercase Convert all text in the component to lowercase.
Alignment Align text left, center, or right using the alignment buttons.

Spacing

Property Description
Line Height Adjust the vertical spacing between lines of text.
Letter Spacing Adjust the horizontal spacing between letters.

Color

Property Description
Text Color Pick a color for the text using the color picker.
Stroke Color & Width Set the outline (stroke) color and width of the text.
Shadow Enable/disable shadow, set shadow color, blur, opacity, and X/Y offsets.

Shadow Blur Adjust the blur radius of the shadow.
Shadow Offset X/Y Move the shadow horizontally (X) or vertically (Y).

Batch Updates

When multiple text components are selected, the text area is hidden and only shared properties like font, color, stroke, and shadow can be updated for all selected components simultaneously.
