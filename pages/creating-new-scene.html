Creating a New Scene

There are several ways to create a new scene. When you first open GDS Editor, a default 1920 Ã— 1080 pixel scene is automatically created. You can start working on this scene immediately, or create a new scene with your own settings. Additionally, if you have a previously saved scene from a GDS file or a GDS file from a downloaded asset pack, you can load it into the editor.

âž• Creating a Scene

You can create a new scene at any time using the following methods:

  • Press the hotkey N to open the New Scene dialog.
  • Use the "Add Scene" button in the Scene Tab panel on the left side of the interface scene tabs #2 .
  • Click the "Create New Scene" button on the Welcome Window when first starting the editor. *Make sure the welcome window is visible if you previously turned it off.
  • Drag a saved GDS scene file into the editor. You will be prompted to either load it as a new scene or merge it into the current scene.

When the New Scene dialog opens, it displays various options for creating a new scene:

1 Load an Existing .gds File If you have a previously saved file or a .gds file from an asset store download, use this option to navigate to the corresponding directory on your disk and load it as a new scene.
2 Create a New Blank Scene You can start a new scene with your own custom dimensions according to your project needs via this form.
3 Preset Scene Size GDS Editor is primarily aimed at game developers, so it includes a list of helpful preset dimensions for creating scenes, such as Steam store promotional materials, itch.io store sizes, and common screen sizes for mobile and desktop applications. Simply click one to start.
If you think of a preset size that would be helpful, let us know on the Discord channel and we will add it.
